About Me:

Hi, I'm Bonnie, a student, gamer, and programmer.

I've always loved videogames and wanted to start making my own. Initially I started learning Java to make mods for Minecraft, but later moved on to learning C# to develop my own games in the Unity Engine.

I released my first game, Obunga, after about a year of development (it took this long because I was lazy) and it was fairly well received. Recently I've gotten back into Java and have been working on a Minecraft mod called The Sideways.
